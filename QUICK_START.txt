TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Problem: "Windows protected your PC" warning
Solution: Click "More info" > "Run anyway" (this is normal for new executables)

Problem: Application won't start
Solution: Try running as Administrator
          → Right-click FastFoodPOS.exe > Run as administrator

Problem: setup.bat gives "Access Denied"
Solution: Run Command Prompt as Administrator
          → Right-click Start > Windows Terminal (Admin)
          → Navigate to folder and run: setup.bat

Problem: Can't find installed application
Solution: Check Start Menu > type "NSA Fast Food"
          or look for desktop shortcut
